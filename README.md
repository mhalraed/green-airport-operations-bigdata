# Green Airport Operations Big Data Project

## 1. Project Overview

This project uses Databricks, PySpark, and Big Data processing techniques to analyse airport operations data and identify patterns related to delays, taxi time, congestion, and avoidable emissions.

The aim of the project is to build a scalable data pipeline and use PySpark-based analytics and machine learning to support greener airport operations.

## 2. Tools and Technologies Used

- Databricks
- Apache Spark
- PySpark
- Hadoop-compatible distributed storage concepts
- Delta Lake
- Spark SQL
- Spark MLlib
- Python
- GitHub
